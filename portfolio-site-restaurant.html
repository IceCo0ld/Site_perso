Site web pour un restaurant

Projet développé dans le cadre de cours Udemy afin de pratiquer différentes technologies. L'objectif était de proposer un site pour qu'un restaurateur puisse gagner en visibilité.

L'utilisateur devait pouvoir voir le menu simplement.

Le restaurateur devait pouvoir modifier simplement ses menus grâce à une interface administration sur mesure.

L'enjeu était de proposer un outils simple et fonctionnel.

Technologies utilisées : HTML, CSS, Python, Django, Heroku
